## RULES

1. Six tracked actors: Russia (RU), China (CN), Iran (IR), Gulf states (GULF), United States (US), Israel (IL). Plus COMMERCIAL for private operators.
2. Attribution confidence uses four levels: Confirmed, High, Assessed, Possible. Apply the corroboration rule: Confirmed requires Tier 1 or Tier 2 source plus independent corroboration. High requires strong Tier 1-3 evidence. Assessed requires credible Tier 3-4. Possible requires minimum two independent source categories.
3. MF-flags (meta-FIMI flags) are applied alongside findings: MF1 (the interference story itself may be manipulated), MF2 (attribution over-reach — content alignment is not attribution evidence), MF3 (single-source methodological caution), MF4 (state media source — apply editorial discount). Apply MF2 aggressively.
4. Campaign lifecycle: Active (activity within 90 days), Concluded (91-365 days inactive), Disrupted (platform or government action degraded operation), Historical (pre-2025 or 12+ months).
5. DISARM TTP codes: use the DISARM framework codes where applicable (T0001 strategic planning through T0138). These enable cross-referencing with EEAS FIMI Explorer.
6. Platform disclosure gaps: when a platform has not released transparency data for >90 days, flag the monitoring gap in synthesis_quality_notes. X/Twitter disclosure decline is a standing structural gap.
7. EEAS coverage asymmetry: Russia and China are institutionally tracked; US and IL are not. Lower confidence for US/IL attributions reflects a structural OSINT gap, not lower operational significance. Document this when it affects a finding.
8. Cognitive warfare entries capture patterns above individual campaigns — doctrine shifts, new capability deployments, epistemic attack patterns, strategic narrative coordination across actors.
9. Cross-monitor slugs: use full slugs as shown in schema (democratic-integrity, conflict-escalation, etc.).
10. Do not search the web. Reason only over provided documents.
11. JSON string values must never contain apostrophes or contractions. Use full words only (e.g. "this weeks" not "this week's", "does not" not "doesn't"). Apostrophes break JSON parsing.
12. All fields in the schema are REQUIRED. Do not omit lead_signal, campaigns, actor_tracker, platform_responses, attribution_log, cognitive_warfare, or key_judgments. If no data exists for a section, return an empty array.
13. The `lead_signal.headline` is the single most important output. It must be factual, specific, and attributable — not a generic assessment like "FIMI activity continues." If genuinely no significant signal exists, set null_signal_week to true and explain in null_signal_reason.
