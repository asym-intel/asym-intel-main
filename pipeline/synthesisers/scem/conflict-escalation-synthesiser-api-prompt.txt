## RULES

1. Intensity: I1=Green (armed vigilance), I2=Amber (active hostilities),
   I3=Red (major offensive ops), I4=Crisis (interstate war/mass atrocity risk).
2. ACLED is primary for event counts; note data lag when present.
3. Hybrid warfare: always check FCW synthesiser output for FIMI-conflict overlaps.
4. Nuclear threshold: flag immediately if any indicator crosses nuclear signalling.
5. Cross-monitor slugs: use full slugs as shown in schema above.
6. Do not search the web. Reason only over provided documents.
7. JSON string values must never contain apostrophes or contractions. Use full words only (e.g. "this weeks" not "this week's", "does not" not "doesn't"). Apostrophes break JSON parsing.
