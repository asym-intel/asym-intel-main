## SEVERITY SCORING RUBRIC

Formula: Score = A + B + C + (2.5 - D)
Final score rounded to nearest 0.5, capped 1-10.

Dimension A — LDI Trajectory: 0 (stable) | 1.0 (gradual decline) | 2.0 (steep) | 2.5 (collapse)
Dimension B — Institutional Breadth (pillars under attack): 0 (0-1) | 1.0 (2-3) | 2.0 (4-5) | 2.5 (all 6)
Dimension C — Repression Severity: 0 (none) | 0.5 (harassment) | 1.0 (arrests) | 2.0 (killings) | 2.5 (mass killings)
Dimension D — Resilience (INVERTED): 0 (none) | 0.5 (weak) | 1.0 (moderate) | 1.5 (significant) | 2.0 (strong) | 2.5 (robust)

Six pillars: judiciary, legislature, civil service, press, civil society, elections.

Status thresholds:
- Rapid Decay: score by rubric; requires 2+ pillars under attack
- Recovery: Rapid Decay country with net positive trajectory
- Watch List: 2+ leading indicators; promoted to Rapid Decay when score > 5.0 on two consecutive weeks

## DATA LIFECYCLE MANAGEMENT

Entry Types:
- Persistent: Structural data (legal frameworks, doctrine shifts) — stays until material change
- Episode: Bounded event sequence — stays until resolution, then archived
- Transient: Single announcement — rolled up once implications captured, never deleted

Confidence Levels:
- Confirmed: 2+ independent T2-T4 sources
- Probable: Single strong T2-T4 source
- Reported: Single T3-T4 source, not corroborated

Pipeline Validation Rule:
- 0 sources + Confirmed/High confidence: Hard reject
- 1 source + Confirmed/High: Auto-downgrade to Moderate; preserve original in confidence_original
- 2+ sources + Confirmed/High: Pass
- Any count + Moderate or below: Pass
