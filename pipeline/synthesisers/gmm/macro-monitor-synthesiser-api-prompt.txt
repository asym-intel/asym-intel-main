Four tiers apply to all main signal confidence fields and key_judgments across all monitors.

| Tier | Label | Corroboration standard |

| 1 | Confirmed | 2+ independent sources, at least one T1-T2. No active contradicting evidence. |
| 2 | High | Single strong T1-T3 source, or 2+ T3-T4 sources consistent. No direct contradiction. |
| 3 | Assessed | Credible T3-T4 evidence. Analytical judgment supported but not independently corroborated. Default for most signals. |

## MATT FRAMEWORK (Positioning Overlay)

MATT = Market Agreement to Our Themes. For each tracked warning theme:
- 0-2 (Very Low): Theme not discussed in mainstream IB research
- 3-4 (Low): Acknowledged but consensus treats as low probability
- 5-6 (Medium): Consensus acknowledges; partial pricing
- 7-8 (High): Theme is consensus view; majority IB coverage
- 9-10 (Fully priced): Mainstream narrative; limited alpha

High MATT = less actionable alpha. Low MATT + strong evidence = most actionable.

## RULES

2. system_average is the arithmetic mean of all indicator domain stress scores: Green=0, Amber=-0.33, Red=-0.66, Crisis=-1.0. Range: 0.0 (all green) to -1.0 (all crisis). Negative = stress. CRITICAL threshold: below -0.50.
3. conviction reflects how many domains corroborate the regime: 1 domain = LOW, 2-3 = MEDIUM, 4-5 = HIGH, 6 = VERY HIGH.
4. lead_signal is REQUIRED. The headline must be factual and specific — not a regime label.

7. Tariff tracker: document all new measures and retaliation chains this week.
8. Central bank tracker: note divergence between major institutions.
9. Null signal: set true only if all six domains Green/stable, no regime delta.
10. Cross-monitor slugs: use full slugs as shown in schema above.
11. Do not search the web. Reason only over provided documents.
12. JSON string values must never contain apostrophes or contractions. Use full words only (e.g. "this weeks" not "this week's", "does not" not "doesn't"). Apostrophes break JSON parsing.
