## CONFIDENCE TIER STANDARD (cross-monitor canonical — v1.0, approved 16 Apr 2026)

Four tiers apply to all main signal confidence fields and key_judgments across all monitors.
Use exactly these labels — no synonyms.

| Tier | Label | Corroboration standard |
|------|-------|----------------------|
| 1 | Confirmed | 2+ independent sources, at least one T1-T2. No active contradicting evidence. Will be rare on this monitor — that is correct. |
| 2 | High | Single strong T1-T3 source, or 2+ T3-T4 sources consistent. No direct contradiction. |
| 3 | Assessed | Credible T3-T4 evidence. Analytical judgment supported but not independently corroborated. Default for most signals. |
| 4 | Possible | Min 2 independent source categories. Pattern-consistent but evidence thin, indirect, or early. |

key_judgments use three tiers only: High | Assessed | Possible
(Key judgments are analytical inferences — Confirmed is reserved for primary signal items.)

Auto-downgrade rule: if confidence == "Confirmed" but research_traceback lacks a T1-T2 source, downgrade to "High".
Cross-monitor flags use a separate scale: Low | Medium | High (unchanged).

## RULES

1. Six domains are primary organising framework. Every development maps to one domain.
2. Four tracked actors: Russia (RU), China (CN), United States (US), Israel (IL). The EEAS tracks only RU and CN — this monitor expands to four per CIDOB gap analysis.
3. HT taxonomy: HT1 links to SCEM, HT3 links to FCW, HT4 links to cyber layer, HT5 links to WDM.
4. US-EU tracker: update every week even if no new developments — carry forward last status.
5. ReArm Europe/EDIP/EDF: track budget commitments, capability milestones, political blockers.
6. State capture risk: tracked for six elevated-risk states (HU, GE, SK, RS, AT, CY). Dimensions: executive capture, judicial independence erosion, media control, foreign financial dependency, veto leverage.
7. Election threat assessment: flag all EU member state elections within 90 days.
8. Lagrange Point scores: five policy vectors measuring EU strategic autonomy progress. Scores are directional indicators, not precision measurements.
9. Attribution confidence uses four levels: Confirmed, High, Assessed, Possible. Apply consistent standards across all four actors. See CONFIDENCE TIER STANDARD above.
10. Source hierarchy: T1 (institutional/diplomatic), T2 (real-time threat data), T3 (investigative/forensic), T4 (infrastructure/technical), T5 (strategic defence research).
11. Cross-monitor slugs: use full slugs as shown in schema above.
12. Do not search the web. Reason only over provided documents.
13. JSON string values must never contain apostrophes or contractions. Use full words only.
14. All fields in the schema are REQUIRED. If no data exists for a section, return an empty array.
