QUIET WEEK HANDLING:

actor_tracker is empty, this is a low-signal week. In this case:
- Set lead_signal.headline to reflect the absence of signal, not a false positive.
- Set lead_signal.confidence to "N/A".
- Set campaigns to [] and actor_tracker entries to status: "MONITORING" with
  a summary noting no new signals this week.
- Set weekly_brief_draft to a structured absence-of-signal assessment
  (see weekly_brief_narrative in the weekly input for reference language).
- Check daily-latest.json and reasoner-latest.json for any findings that
  elevate the picture — if so, lead with those.
- Do NOT synthesise a false lead signal from background context.

  pipeline this week. The lead signal must be specific: name the actor, operation,
  and target. Do not select a "general trend" — pick one concrete event or finding.
  On a quiet week, the lead signal is the absence of signal (see above).

B. KEY JUDGMENTS
  Produce 2–5 judgments. Each must be:
  — A single declarative sentence (claim + confidence tag)
  — Grounded in evidence from the pipeline documents
  — Actionable for a policy or security audience
  Confidence tags: [Confirmed] [High] [Assessed] [Possible]
  On a quiet week, judgments may cover structural gaps, actor dormancy, or
  monitoring limitations — these are analytically valid conclusions.

C. CAMPAIGN SUMMARIES
  For each campaign in weekly-latest.json and any continuation findings in
  daily-latest.json:

  — Set severity: CRITICAL | HIGH | MODERATE | LOW
  — Incorporate any reasoner linkage detections or attribution upgrades/downgrades
  — Populate delta_note: what specifically changed this week vs last week
  If reasoner recommends a confidence change, adopt it with a note
  (Analyst will confirm or override).

D. ACTOR TRACKER
  Synthesise posture for all six actors (RU, CN, IR, Gulf, US, IL).
  Each entry must include:
  — status: HIGHLY ACTIVE | ACTIVE | MONITORING | NO SIGNAL

  — headline: most significant development this week (or "No signal this week")
  — summary: 1–2 sentence analytical note (REQUIRED — do not leave empty)

  On a quiet week, all actors default to MONITORING with explicit absence note.

E. INTELLIGENCE HIGHLIGHTS

  — Platform enforcement actions
  — Contested findings flagged by the reasoner that need Analyst review
