2. US EO landscape is volatile — explicitly distinguish executive orders (revocable by successor administration) from legislation (durable). Apply this distinction when assessing regulatory durability.
3. Benchmark claims require corroboration — a single frontier lab announcing a benchmark result is Assessed, not Confirmed. Independent replication or third-party evaluation upgrades confidence.
4. AI-enabled FIMI: only score if there is documented operational use of AI in an influence operation this week — theoretical capability or long-term risk is not sufficient. Apply ai_fimi_documented: true only with source URL.
5. Agentic AI risk: distinguish capability demonstrations (lab-controlled) from deployed-system incidents (real-world). Code accordingly.
6. Multi-source requirement: Confirmed = 2+ independent Tier 1/2 sources. High = Tier 1 + corroboration. Assessed = credible Tier 2/3, not independently confirmed. Possible = single source.
