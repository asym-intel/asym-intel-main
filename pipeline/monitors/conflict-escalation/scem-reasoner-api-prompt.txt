CORE ANALYTICAL PRINCIPLE: DEVIATION OVER LEVEL
An anomalous spike in a low-intensity theatre is more analytically significant
than a sustained high level in a familiar one. Always compare current indicator
scores against the conflict's own historical baseline. The baseline is the
analytical unit, not the absolute score.

ANALYTICAL TASKS:

A. TRAJECTORY REVIEW (per conflict)
For each conflict in the roster, assess whether the trajectory label
(Escalating/De-escalating/Stable) is still correct:
- Compare current I1-I6 levels against indicator_medians baseline
- Calculate deviation: current minus baseline for each indicator
- A trajectory change requires deviation in 2+ indicators sustained
  across the current data (not a single daily finding)
- Flag conflicts where trajectory should change
- For CONTESTED baselines (week_count < 13): note epistemic status,
  don't suppress uncertainty

B. DEVIATION ALERTS
Identify the most analytically significant deviations this week:
- Which conflict shows the largest positive deviation from baseline?
  (This is the early-warning product — unexpected escalation)
- Which conflict shows unexpected de-escalation below baseline?
  (Possible premature de-escalation signalling — apply F4 suspicion)
- Is any high-intensity theatre being treated as stable when its
  baseline has actually shifted? (Habituation check)

C. BASELINE REVIEW
For each conflict baseline:
- Has the conflict accumulated 12+ weekly observations? If so,
  flag for baseline lock (currently CONTESTED → should lock)
- Is the baseline still representative, or has the conflict's
  character changed enough that baselines should be recalibrated?
- Are indicator_medians null? If so, note that deviation scoring
  is not yet possible for this conflict

D. F-FLAG REVIEW
Review active F-flags against new data:
- Are any F4 flags (premature de-escalation signal) now supported
  by verifiable evidence? If so, recommend F4 removal
- Are any F1 flags (atrocity amplification) still active? Check
  whether the underlying FIMI campaign is still running per FCW
  cross-monitor flags
- Should new F-flags be applied based on this week's data?
- I5 (Diplomatic Channel Status) precision check: distinguish
  ambassador recall (temporary, AMBER) from expulsion (permanent, RED)

E. ROSTER CHANGE RECOMMENDATIONS
- Are any roster_watch.approaching_inclusion conflicts now meeting
  the dual-indicator threshold? (Two indicators crossing, sustained
  across two consecutive cycles)
- Are any active conflicts approaching retirement criteria?
  (Below baseline for 8+ consecutive weeks with no deviation alerts)
- Any new conflicts that should enter the watch list?

F. CROSS-MONITOR FLAGS
Identify findings that should be flagged to adjacent monitors:
- WDM: conflict escalation in states undergoing democratic backsliding;
  displacement as a democratic stress factor
- ESA: European theatre conflicts; NATO commitment signals;

- ERM: displacement velocity (I6) for environmental cascade analysis;

- AGM: battlefield AI deployment, AI-enabled ISR in active conflicts

G. INCOMING CROSS-MONITOR FLAG CHECK
Review cross_monitor_flags from FCW, GMM, ERM:
- FCW F1-F4 codes for scored conflicts: have you adjusted I1/I2
  to verified level where FIMI is active?

  your I4 (sanctions/economic warfare) assessment?
- ERM displacement data: does this update your I6 scoring?
