Do not allow a week to pass without at least one qualifying finding per active theatre even if the trajectory is improving.

SIX INDICATORS (I1-I6) — apply to every finding:
I1 Rhetoric Intensity — statements, declarations, threats, posturing
I2 Military Posture — troop movements, operations, equipment transfers, territorial changes
I3 Nuclear/Strategic — doctrine changes, strategic weapon deployments, nuclear signals
I4 Economic Warfare — sanctions, embargoes, resource coercion, financial pressure
I5 Diplomatic Channels — negotiations, ceasefires, ambassador recalls/expulsions, multilateral engagement
I6 Displacement Velocity — IDP/refugee flow changes, humanitarian corridor status, crisis deterioration

INDICATOR CALIBRATION — CRITICAL:
- Drone warfare is BASELINE in Russia-Ukraine and Gaza. Do NOT score standard drone operations as I2 escalation. Only score: confirmed range extension beyond prior limits, swarm doctrine first use, or new drone type entering theatre.
- Russia Nov 2024 nuclear doctrine shift is ALREADY BASELINE for I3. Subsequent Medvedev or Kremlin rhetoric on nuclear use is episodic — set episodic_flag: true and do not re-score as escalation.
- US diplomatic channels are structurally de-weighted for 2026 — US-brokered ceasefire proposals carry lower I5 weight than in prior years. Apply this discount explicitly in confidence_basis where relevant.
- Ambassador recall ≠ expulsion — code differently. Recall = I5 downgrade; Expulsion = I5 escalation.

ANALYTICAL RULES:
1. Multi-source requirement: findings require confirmation from at least 2 independent sources before Confirmed or High confidence. Single-source findings = Assessed or Possible.
2. AI-generated propaganda: apply aiGenerationCheck = true for I1 findings sourced from unofficial channels; do not score as Confirmed.
3. Sudan/Myanmar/DRC structural gap: flag any week where fewer than 3 combined findings come from these theatres — this is a coverage failure, not an absence of events.
4. Episodic vs. structural: tag episodic_flag: true for one-off statements, single incidents, or events without clear trajectory signal. Only sustained patterns across 3+ reporting cycles upgrade to structural.

OPEN-ENDED COLLECTION (REQUIRED): After completing all structured sections above,
run one additional broad search: "what else happened this week in conflict, strategic stability, or escalation dynamics that a
sophisticated analyst would flag as significant?" Return findings as open_findings[]
with no schema constraint beyond the fields shown. Minimum 1 entry, no maximum.
This is not a catch-all for things that didn't fit above — it is a deliberate
breadth pass to surface novel signals the structured schema may not have anticipated.

REVERSE-CASCADE CHECK (REQUIRED): Populate reverse_cascade_check at the top level of the output.

- cascade_description: a 1-2 sentence description of the cascade mechanism (null if checked is false).
- geopolitical_event: short identifier of the event (null if checked is false).
- note: mandatory analyst summary note (1 sentence minimum). When no qualifying event is identified this cycle, set checked: false and note: "No reverse-cascade event identified this cycle."
