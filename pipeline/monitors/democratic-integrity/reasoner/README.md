# WDM Reasoner Output

**Monitor:** World Democracy Monitor (democratic-integrity)
**Layer:** Reasoner — Deterioration Pattern Analysis (GitHub Actions)
**Model:** sonar-deep-research (reasoning over structured data — NO web search)
**Schedule:** Sunday 20:00 UTC (2 hours after weekly research, 10 hours before Analyst)

## Files

- `reasoner-latest.json` — most recent reasoning run (overwritten weekly)
- `reasoner-YYYY-MM-DD.json` — dated archive

## Purpose

Deep analytical reasoning over accumulated WDM structured data:
- Deterioration trajectory analysis across active cases
- Mimicry chain linkage detection (shared legislative templates)
- Country-specific confidence review (upgrade/downgrade recommendations)
- Cross-monitor escalation flags (SCEM, FCW, ESA linkages)
- Contested findings requiring human review

The WDM Analyst reads this at Step 0E each Monday for analytical
pre-work before applying the full scoring methodology.

## Schema

`reasoner-v1.0`

## Critical note

sonar-deep-research does NOT search the web. It reasons only over
structured JSON context provided inline in the prompt. This is correct —
the Reasoner is a synthesis layer, not a collection layer.
