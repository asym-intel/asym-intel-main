# docs/seo/ — SEO & Discoverability Audit Files

This directory contains quarterly GSC audit reports for asym-intel.info.

## Files

- `gsc-latest-audit.md` — always the most recent audit (overwritten each quarter)
- `gsc-quarterly-YYYY-QN.md` — archived quarterly reports

## Schedule

Quarterly audits run automatically on: 1 April, 1 July, 1 October, 1 January.
First audit: 1 July 2026 (Q2 2026 — first full quarter of platform operation).

The SEO & Discoverability Expert reads `gsc-latest-audit.md` at Step 0 of
any SEO session. See `docs/prompts/seo-discoverability-expert.md`.
