# Environmental Risks Monitor (ERM) — Asymmetric Intelligence

## Focus
Climate tipping points, biodiversity loss, resource scarcity, environmental policy

## Publication cadence
Weekly. Publishes Saturday 09:00 UTC.

## Data endpoints
- Latest report (JSON): https://asym-intel.info/monitors/environmental-risks/data/report-latest.json
- Latest report (Markdown): https://asym-intel.info/monitors/environmental-risks/data/report-latest.md
- Monitor page: https://asym-intel.info/monitors/environmental-risks/

## JSON report structure
The report-latest.json file contains:
- `meta`: issue number, volume, week label, published timestamp, schema version
- `summary`: executive summary of the week's key developments
- `structured_claims[]`: array of verifiable claims with confidence labels and evidence IDs
- `persistent_state`: standing knowledge base — entities, ratings, tracked vectors
- `_meta`: pipeline metadata (cycle ID, methodology version, composed_at)
