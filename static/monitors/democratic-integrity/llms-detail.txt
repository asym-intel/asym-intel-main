# World Democracy Monitor (WDM) — Asymmetric Intelligence

## Focus
Electoral integrity, judicial independence, civil society, press freedom

## Publication cadence
Weekly. Publishes Monday 06:00 UTC.

## Data endpoints
- Latest report (JSON): https://asym-intel.info/monitors/democratic-integrity/data/report-latest.json
- Latest report (Markdown): https://asym-intel.info/monitors/democratic-integrity/data/report-latest.md
- Monitor page: https://asym-intel.info/monitors/democratic-integrity/

## JSON report structure
The report-latest.json file contains:
- `meta`: issue number, volume, week label, published timestamp, schema version
- `summary`: executive summary of the week's key developments
- `verified_claims[]`: array of verifiable claims with confidence labels and evidence IDs
- `persistent_state`: standing knowledge base — entities, ratings, tracked vectors
- `_meta`: pipeline metadata (cycle ID, methodology version, composed_at)
