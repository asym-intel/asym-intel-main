# FIMI & Cognitive Warfare Monitor (FCW) — Asymmetric Intelligence

## Focus
Foreign information manipulation, cognitive warfare campaigns, platform responses

## Publication cadence
Weekly. Publishes Wednesday 06:00 UTC.

## Data endpoints
- Latest report (JSON): https://asym-intel.info/monitors/fimi-cognitive-warfare/data/report-latest.json
- Latest report (Markdown): https://asym-intel.info/monitors/fimi-cognitive-warfare/data/report-latest.md
- Monitor page: https://asym-intel.info/monitors/fimi-cognitive-warfare/

## JSON report structure
The report-latest.json file contains:
- `meta`: issue number, volume, week label, published timestamp, schema version
- `summary`: executive summary of the week's key developments
- `structured_claims[]`: array of verifiable claims with confidence labels and evidence IDs
- `persistent_state`: standing knowledge base — entities, ratings, tracked vectors
- `_meta`: pipeline metadata (cycle ID, methodology version, composed_at)
